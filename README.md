# Mini-projet : Employabilité des étudiants

## Contexte
Ce mini-projet a été réalisé dans le cadre du cours de statistique à l'ENSA Kenitra. 
Il vise à analyser les facteurs influençant l'employabilité des étudiants en Génie Industriel et Informatique.

## Contenu du dépôt
- **Rapport_Final.pdf** : Rapport final en anglais du projet
- **Script_R.R** : Script R utilisé pour l'analyse statistique

## Auteur
- Moulaye Abdoule Hady Haidara et coéquipiers

## Instructions
Ouvrir `Script_R.R` dans RStudio pour reproduire les analyses. Le rapport `Rapport_Final.pdf` contient toutes les conclusions et visualisations.
